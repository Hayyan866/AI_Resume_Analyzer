# AI Resume Analyzer

AI Resume Analyzer is a web app that analyzes resumes using AI, extracts skills, experience, and education, and provides suggestions to improve resume quality and ATS compatibility.

## Features
- Upload and analyze resumes
- Extract skills, experience, and education
- Identify missing skills for target jobs
- Resume scoring and improvement suggestions
- Interactive dashboard with skill insights
- Secure login and signup

## Tech Stack
- Frontend: React, TypeScript, Tailwind, Vite
- Backend: Node.js, Express
- Database: PostgreSQL / MongoDB
- AI/NLP: OpenAI API

## Installation
```bash
git clone https://github.com/Hayyan866/AI_Resume_Analyzer.git
cd AI_Resume_Analyzer
# Install frontend
cd frontend
npm install
# Install backend
cd ../backend
npm install
# Running
# Start backend
cd backend
npm run dev
# Start frontend
cd ../frontend
npm run dev
