# Welcome to your project

## Project info

**Name**: AI Resume Analyzer
**Description**: An AI-powered resume analyzer and job match platform.

## How can I edit this code?

This project is built using:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Running Locally

You will need Node.js & npm installed.

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server with auto-reloading and an instant preview.
npm run dev
```

### Backend

The backend is built with Node.js and Express, available in the `backend` directory.

To run the backend:

```sh
cd backend
npm i
npm run dev
```
